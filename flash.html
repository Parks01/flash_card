Learn JavaScript and jQuery!

Terms and Concepts to learn:

JavaScript(JS)

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operator

arithmetic symbols to perform arithmetic on numbers and variables.

Variables

Variables are assigned to strings, numbers, etc. to refer back to the data.

Variable naming convention

Can use letters, certain symbols (#, _, -), camelCase.

Functions

Nested action(s) within { } that can be called and "saved for later use."

Methods

Actions that can be performed on objects. It is a property that contains a function definition.

Arguments

Content that replaces the parameter when calling the function.

Parameters

What is initially defined in the function(inHere).
